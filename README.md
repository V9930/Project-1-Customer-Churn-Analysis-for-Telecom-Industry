# Project-1-Customer-Churn-Analysis-for-Telecom-Industry

Project Overview

This project analyzes customer churn behavior in a telecom company using data analysis and visualization techniques. The goal is to identify patterns and key factors that influence customer churn and provide insights that can help businesses reduce customer loss.

Objective

- Analyze telecom customer data to understand churn patterns.
- Identify factors such as contract type, internet service, tenure, and monthly charges that influence churn.
- Build an interactive dashboard to visualize customer behavior and churn trends.

Tools & Technologies

- Python (Pandas, Matplotlib, Seaborn)
- SQL (PostgreSQL)
- Power BI
- Excel / CSV Dataset

Project Workflow

1. Data Collection

The dataset contains telecom customer information such as:
CustomerID, Gender, SeniorCitizen, Partner, Dependents, Tenure, PhoneService, InternetService, Contract, PaymentMethod, MonthlyCharges, TotalCharges, and Churn.

2. Data Cleaning (Python)

- Handled missing values
- Converted data types
- Cleaned and prepared dataset for analysis

3. Exploratory Data Analysis (EDA)

Used Python libraries to analyze patterns and relationships:

- Churn distribution
- Churn by contract type
- Churn by internet service
- Monthly charges vs churn
- Tenure distribution

4. SQL Analysis

Used SQL queries in PostgreSQL to extract business insights:

- Total number of customers
- Total churned customers
- Churn rate
- Churn by contract type
- Churn by internet service
- Average monthly charges by churn

5. Power BI Dashboard

Created an interactive dashboard to visualize key insights including:

- Total customers
- Churn rate
- Churn distribution
- Churn by contract type
- Churn by internet service
- Monthly charges vs churn
- Tenure distribution

Key Insights

- Customers with month-to-month contracts have the highest churn rate.
- Customers using fiber optic internet service show higher churn compared to other services.
- Customers with higher monthly charges are more likely to churn.
- Many customers leave the service within the first year of subscription.

Conclusion

This project demonstrates how data analysis can help telecom companies understand customer behavior and reduce churn by identifying high-risk customer segments.
